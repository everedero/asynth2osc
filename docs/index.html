aSynth2OSC turns your modular gestures and modulations into show control.

Built for artists and technicians who want to get some expression from analog synthetizer rig across sound, lights, and visuals.

aSynth2OSC Eurorack Module bridges analog CV, triggers, cue remote and MIDI to robust OSC over Ethernet.

Send those to your favorite show environment like Open Stage Control, Chataigne, Max/MSP, PureData, TouchDesigner, SuperCollider, Jitter, Resolume, Notch, Reaper, QLab etc. and perform with tactile immediacy, without the need for a computer on stage.

What It Does On Stage

aSynth2OSC is not just a converter. It is a performance bridge: tactile modular control in front, deterministic OSC workflows in the back. Every modulation can become a color, an intensity curve, a video state, or a spatial path position.

Cue Recall Manager

Recall cues to change context on remote software or hardaware, with immediate Prev/Next navigation or with rotary encoder to jump to a specific cue.

Modular Dynamics To OSC

Convert 4 CV channels to normalized float values, with adjustable period and hysteresis, ideal for smooth automation lanes in media engines.

Two-Way Feedback

Receive incoming OSC commands to show confirmations directly to the performer OLED screen. You can also set the CV Analog output and flash buttons LEDs if needed.

Creative Use Cases

Typical integrations in contemporary live setups where music, visual systems, and stage automation are tightly linked.

1) Control Snapshot Recall Across Systems

Use the encoder to select cue 128, validate, and instantly recall matching states and mappings in your lighting console, video server, and audio engine.

Cue 128 -> /cue 128 -> lights preset + media scene + FX routing

2) Animate Sounds, Lights And Video With CV

Patch an LFO or envelope into CV 1 jack input and map normalized OSC floats to beam spread, strobe depth, shader intensity, or playback speed.

/cv/1 0.00..1.00 -> DMX intensity curve / video shader gain

3) Move Sound Objects In Spatial Audio

Drive object (X,Y) coordinates from CV joystick pairs and trigger snapshots for trajectory changes, making movement feel physically performed instead of programmed.

CV1 -> X axis, CV2 -> Y axis, Trig1 -> start animation

4) Send hardware sequencer's MIDI from stage to FOH

Forward MIDI PC #7 sequencer song pointer into OSC pathways so context stay in sync with external show controllers and render engines by recalling their snapshots.

/program 1 7 -> timeline jump / scene alignment to program 7

Built For Real-World Operation

Operational Strengths

  • Instant boot and send OSC, persistent settings.
  • RJ45 wired network interface. Static or DHCP addressing with editable subnet bytes.
  • On-device inputs monitoring with activity and levels indicators.
  • Simple ping/pong and messenger commands for remote diagnostics.

User Manual (Quick Reference)

A practical end-user workflow for powering on, recalling cues in performance mode, entering Settings mode, and safely returning to performance mode.

1) How To Start

  1. Power the module and wait for splash screen completion.
  2. Confirm you are in normal mode (center label shows Recl).
  3. Check network reachability from your show computer.
  4. Send a /ping to verify OSC round-trip with /pong.

2) Normal Mode: Show Control

  1. Press Prev/Next to recall cues immediately (000..999 wrap).
  2. Turn encoder to prepare a pending cue (blinking value).
  3. Short press encoder to confirm and send the cue.
  4. If not confirmed in 5 seconds, pending cue is canceled.
  5. Short press with no pending edit resends current cue.

3) Settings Mode: Menu Navigation

  1. Long press encoder 2 seconds to enter menu mode.
  2. Use Prev/Next to move through menu items.
  3. Turn encoder to change selected value.
  4. Short press encoder resets current item to default. (see table below)
  5. Long press encoder 2 seconds to exit back to Show Control mode.

Hardware, UI & Connections

14 HP Eurorack standard module. Powered by +12V, -12V from the Eurorack standard bus or 5V USB dedicated power supply (estimated 250mA).

Display and controls

The screen shows live performance data and navigation:

  • Cue Number (top left): 3-digit value (000–999) is the current cue number. Blinks when recall is pending a confirmation by pushing the encoder. When in menu mode, edit the setting value
  • Activity Meters (top right): "1 M A 2" row displays blinking indicators for Trigger 1, Midi in activity, Analog out, Trigger 2 and four live CV level bars
  • Messenger Line (center left): Shows transient messages received from OSC /msg. When in Menu mode, displays the current setting name and value
  • Button Labels (botom): "Prev", "Recl" (center mode indicator, rotary encoder push), and "Next" align with the physical buttons below the display
  • Rotary Encoder (center below screen): Navigate cues (turn), confirm selections (short press), or switch modes (2-second hold)
  • Prev/Next Buttons: Physical pushbuttons below the display for immediate cue recall (Prev/Next) or navigation in Menu mode

Connections to other modules

Eight 1/8" TRS jacks provide all signal I/O pathways:

  • Trigger 1 & Trigger 2 (top left/right): Gate and trigger inputs for clock pulses, envelopes, or digital state changes. And, yes, they have their own dedicated buttons also for manual finger triggering
  • MIDI IN (top center-left): MIDI jack Type A connector for Note, CC, Program Change, Pitch Bend, transport, Song Position, and MTC conversion to OSC
  • CV OUT (top center-right): Analog Output jack for sending to external modules or for feedback pathways
  • CV 1, CV 2, CV 3, CV 4 (bottom row): Four analog control voltage inputs with 16-bit precision, selectable range (±5V or ±12V), polling at configurable intervals (10–200 ms)

Settings Menu Reference

Enter menu mode by holding the rotary encoder for 2 seconds (center label switches from Recl to Menu). Use Prev/Next to navigate items, turn the encoder to change a value, short press to reset to default, long press 2 seconds to exit. All changes are saved immediately to persistent storage. Network settings take effect only when exiting the menu.
